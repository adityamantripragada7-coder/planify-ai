# planify-ai


A modern SaaS-style landing page for an AI-powered study planner.  
Built using HTML, CSS, and JavaScript with a focus on clean UI, responsiveness, and interactive user experience.

---

## ✨ Features

- 🔥 Modern gradient hero section
- ⚡ Smooth scrolling navigation
- 🎯 Interactive UI with animations
- 📊 Feature showcase section
- 📩 Email signup with validation
- 🌐 Fully responsive design
- 🎨 Glassmorphism UI effects
- 💡 Scroll-based animations (Intersection Observer)

---

## 🛠️ Tech Stack

- HTML5 (Semantic Structure)
- CSS3 (Flexbox, Animations, Glassmorphism)
- JavaScript (DOM, Events, API Simulation)

---

## Importance 
A modern SaaS landing page built using HTML, CSS, and JavaScript.
